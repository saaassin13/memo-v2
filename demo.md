# 备忘录 App 页面设计

## 整体结构

底部固定导航栏：应用 | Todo | 日历 | 我的

---

## 页面一：应用（首页）

底部导航第一个 Tab，展示功能入口网格。

| 模块 | 功能 |
|------|------|
| 顶部 | 搜索图标、设置图标 |
| 功能网格 | 2列网格展示功能入口：备忘录、倒数纪念日、日记、记账、目标、体重等 |
| 功能卡片 | 图标 + 名称，点击进入对应功能 |

### 子页面：备忘录列表
从「应用」进入备忘录功能后的列表页。

| 模块 | 功能 |
|------|------|
| 顶部导航 | 「备忘录」标题、搜索图标 |
| 分类标签栏 | 横向滚动，显示「全部/工作/生活/学习」等 |
| 记录列表 | 标题、摘要、日期、置顶标记 |
| 底部 | 「+」新建按钮 |

### 子页面：新建/编辑记录

| 模块 | 功能 |
|------|------|
| 顶部导航 | 返回、标题输入框、保存按钮 |
| 内容编辑区 | 标题输入框、多行文本输入框 |
| 底部工具栏 | 分类选择、提醒时间、置顶开关 |
| 自动保存 | 草稿自动保存 |

### 子页面：日记

| 模块 | 功能 |
|------|------|
| 顶部导航 | 「日记」标题、写日记按钮 |
| 日历视图 | 月历，点击日期查看当天日记 |
| 今日日记 | 显示当天日期天气、日记内容 |
| 历史日记 | 按日期列表展示 |

### 子页面：倒数纪念日

| 模块 | 功能 |
|------|------|
| 顶部导航 | 「倒数纪念日」标题、添加按钮 |
| 纪念日列表 | 名称、目标日期、倒计时天数 |
| 分类 | 生日、节日、重要日等 |

### 子页面：记账

| 模块 | 功能 |
|------|------|
| 顶部导航 | 「记账」标题、统计按钮 |
| 记账输入 | 金额、分类（餐饮/交通/购物等）、备注 |
| 账单列表 | 按日期分组，显示收支明细 |
| 本月统计 | 收入、支出、结余 |

### 子页面：目标

| 模块 | 功能 |
|------|------|
| 顶部导航 | 「目标」标题、添加按钮 |
| 目标列表 | 目标名称、进度条、截止日期 |
| 进度追踪 | 勾选完成进度 |

### 子页面：体重

| 模块 | 功能 |
|------|------|
| 顶部导航 | 「体重」标题、图表按钮 |
| 体重记录 | 今日体重输入、历史记录列表 |
| 趋势图表 | 折线图展示体重变化 |

---

## 页面二：Todo

底部导航第二个 Tab，任务管理。

| 模块 | 功能 |
|------|------|
| 顶部导航 | 「Todo」标题、筛选/排序图标 |
| 分类标签栏 | 横向滚动：「全部」「工作」「生活」「学习」「杂项」「+」 |
| 待办列表 | 任务标题、截止时间、分类标签、勾选框 |
| 已办列表 | 已完成任务（可折叠显示） |
| 右下角 | 「+」新增 Todo 按钮 |

### 新增/编辑 Todo 弹窗

| 模块 | 功能 |
|------|------|
| 任务标题 | 输入框 |
| 截止时间 | 日期时间选择器 |
| 分类选择 | 工作/生活/学习/杂项 |
| 备注 | 可选填写 |
| 保存/取消 | 按钮 |

---

## 页面三：日历

底部导航第三个 Tab，日历视图。

| 模块 | 功能 |
|------|------|
| 顶部模式切换 | 「日」「周」「月」三个切换按钮 |
| 日视图 | 当日时间轴，展示各时段事件 |
| 周视图 | 7天横向展示，每天事件列表 |
| 月视图 | 月历格子，显示每天事件标记点 |
| 今日事件 | 选中日期下方的信息卡片：备忘、纪念日、Todo、目标等 |
| 信息设置 | 可选择显示哪些类型的信息 |
| 右下角 | 「+」新增按钮（可选择 Todo/备忘/纪念日） |

### 信息卡片内容
- 备忘：标题预览
- 纪念日：名称 + 倒计时
- Todo：待办任务
- 目标：目标进度
- 体重：当日体重记录

---

## 页面四：我的

底部导航第四 Tab，个人中心。

| 模块 | 功能 |
|------|------|
| 头像区域 | 头像图片、用户名、编辑按钮 |
| 统计概览 | 记录总数、连续天数等 |
| 功能列表 | 数据导出、清除缓存、主题设置 |
| 关于 | 版本号、反馈入口 |
| 设置入口 | 各项设置入口 |
