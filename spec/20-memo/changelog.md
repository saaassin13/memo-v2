# 20-memo 变更记录

## [1.0.0] - 2026-03-19

### 新增
- MemoListScreen 备忘录列表页面
  - 分类筛选（全部/工作/生活/学习）
  - 列表/网格视图切换
  - 按创建时间/修改时间排序
  - 置顶分组显示
  - 搜索功能
- MemoEditScreen 新建/编辑备忘录页面
  - 标题输入
  - 分类选择
  - 内容编辑
  - 保存逻辑
  - 未保存更改提示
- MemoDetailScreen 备忘录详情页面
  - 完整内容展示
  - 时间戳显示
  - 置顶切换
  - 编辑入口
  - 删除功能（带确认）
- DisplaySettingsSheet 显示设置组件
  - 视图模式切换
  - 排序方式切换

### 路由
- `/apps/memo` - 备忘录列表
- `/apps/memo/new` - 新建备忘录
- `/apps/memo/:id` - 备忘录详情
- `/apps/memo/:id/edit` - 编辑备忘录
