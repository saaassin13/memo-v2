# 11-todo 变更记录

## [1.0.0] - 2026-03-19

### 新增
- TodoScreen 页面框架，包含 Header 和分类筛选
- CategoryFilter 组件：横向滚动分类选择器（全部/工作/生活/学习/杂项）
- TodoListSection 组件：待办列表展示，支持折叠已完成项
- TodoEditSheet 组件：底部弹窗，支持新建/编辑待办
  - 标题输入
  - 分类选择
  - 截止日期选择
  - 备注输入
- 滑动删除功能：Dismissible 实现，带删除确认对话框
- 空状态展示：暂无待办提示，带"添加待办"引导按钮
- 集成 todoProvider 实现数据持久化

### 功能特性
- 分类筛选：支持按分类过滤待办列表
- 待办排序：按截止日期排序
- 已完成折叠：已完成列表默认折叠，可展开/收起
- 完成切换：点击复选框切换完成状态
- CRUD 操作：完整的新建/编辑/删除功能
