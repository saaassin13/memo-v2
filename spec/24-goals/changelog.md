# 24-goals 变更记录

## [1.0.0] - 2026-03-19

### 新增
- GoalsListScreen 页面
  - 目标列表展示（按进行中/已完成分组）
  - 分类过滤（全部/进行中/已完成）
  - FAB 按钮添加新目标
  - 滑动删除功能
- GoalDetailScreen 页面
  - 大号进度环展示
  - 目标详情信息
  - 进度更新（增/减/直接设置）
  - 快速增加按钮（+1/+5/+10）
  - 标记完成/取消完成
  - 编辑和删除功能
- GoalCard 组件
  - 卡片样式，显示目标类型图标和颜色
  - 进度条动画
  - 进度文字和单位
  - 截止日期显示（颜色根据紧急程度变化）
  - 滑动删除
- ProgressRing 组件
  - 环形进度指示器
  - 百分比显示
  - 可自定义大小、颜色、线宽
  - MiniProgressRing 小尺寸变体
- GoalEditSheet 组件
  - 底部弹出表单
  - 名称和描述输入
  - 目标类型选择（每日/每周/每月/每年/自定义）
  - 目标值和单位设置
  - 开始日期和截止日期选择
  - 支持新建和编辑模式

### 路由配置
- `/apps/goals` - 目标列表页
- `/apps/goals/:id` - 目标详情页
